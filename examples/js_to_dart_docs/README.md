# js_to_dart_docs

This example showcases how to use the runtime with `Typescript` and `Node Modules`.
It also showcases how to generate a Dart file from a Typescript file while preserving the documentation.

## Install the runtime

- `dart pub global activate globe_cli`

- `globe runtime install`

## Run the code

- `pnpm install`

- `pnpm run generate`

- `dart run lib/js_to_dart_docs.dart`
